Work

You can find my projects that I worked on here. I've worked on a variety, including cleaning and transforming raw housing data using SQL for in-depth analysis. Additionally, I explored COVID-19 vaccination data from https://ourworldindata.org/, focusing on death percentages in countries like Algeria, USA, and worldwide, along with infection rates.

I utilized Python Pandas for web scraping to track sales on gloves I wanted, creating an automated email notification system for sales. Excel was employed to craft an interactive dashboard showcasing statistics from a coffee shop's sales. I extended the SQL COVID project using Tableau for deeper insights. Moreover, I used POWERBI to analyze Coca-Cola sales, presenting the data in easily comprehensible visualizations.
